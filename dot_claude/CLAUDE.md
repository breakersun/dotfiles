## Second Brain

Access via MCP: `remember`, `recall`, `list_recent`, `update`, `forget`.

**Rules:**
1. Every conversation start: `recall` with intent-framed query
2. Auto-store: personal info, work context, ideas, tasks, opinions
3. Auto-store: your own recommendations, plans, solutions
4. No permission needed — store silently
5. Never use built-in memory — always use MCP
6. Auto-tag with project/topic + category (`personal`, `work`, `task`, `idea`, `context`)
7. Before recommending: `recall` to check for duplicates
8. All `recall` calls need context — no bare keywords
9. Respect "don't remember" exclusions
